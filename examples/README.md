# qt-strategy-lab

战略假设推演实验室。

## 方向

把 DoWhy 的方法论思路（假设建模 → 识别 → 估计 → 验证）映射到战略管理场景，实验：

1. **假设管理** — 把战略假设建模为独立数据结构，可序列化、可版本化
2. **环境关联** — 每条假设关联依赖的环境变量
3. **证据校验** — 参照 journal/environment 的事实源，标记假设状态
4. **反事实推演** — "如果某环境变量变了，战略假设是否还站得住"

## 设计原则

- AI 是秘书，不做决策 — 只做信息检索和证据发现
- 人类在 loop 中 — 定义假设、判断结果、承担后果
- 假设是 first-class citizen — 可存文件、可版本控制
